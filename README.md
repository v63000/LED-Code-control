1. 系统简介
LED Pro V18 是一款轻量级的本地视频播控系统，专为 LED 大屏、展厅投影和数字标牌设计。它允许用户通过手机、平板或电脑的浏览器，远程控制主机的视频播放、音量调节、素材管理及屏幕画面监视。
2. 快速开始
启动软件：运行 Python 脚本。
连接控制端：
确保您的手机/电脑与运行软件的主机在同一局域网（WiFi）下。
打开浏览器，输入主机 IP 地址加端口（默认为 8080），例如：http://192.168.1.100:8080。
界面概览：
播控 (首页)：包含播放控制、实时画面监视、播放列表。
素材 (Tab 2)：视频文件管理、文件夹管理、上传视频。
背景 (Tab 3)：设置视频停止时显示的待机图片（KV图）。
3. 功能详解
3.1 播放控制（首页）
播放/暂停/停止：基础控制。点击“停止”会自动切换显示待机背景图。
音量调节：拖动滑块调节主机音量，点击喇叭图标可一键静音。
播放模式：
🔁 列表循环：播完列表最后这一个后，自动回到第一个播放。
🔂 单曲循环：重复播放当前视频。
🔀 顺序播放：播完列表后停止（显示背景）。
🛑 播完即停：当前视频播完后立即停止。
实时监视 (LIVE)：网页上会实时显示大屏当前的画面（约 10fps），方便盲操作。
投屏目标：如果主机连接了多个显示器，点击数字（1#、2#...）可将画面瞬间切换到对应屏幕。
3.2 播放列表管理（首页下方）
添加视频：在“素材”页面点击视频旁边的 + 号。
排序：长按并拖动列表中的视频条目，可以调整播放顺序。
删除：点击垃圾桶图标将视频从播放队列移除（不会删除源文件）。
清空：点击列表右上角的“清空”按钮移除所有任务。
3.3 素材库管理（素材页面）
浏览：支持多级文件夹浏览。
上传：
点击“上传视频”按钮，选择本地文件（支持多选）上传至当前目录。
支持格式：mp4, avi, mkv, mov, flv, ts。
文件操作：
📷 截取封面：播放视频预览，点击相机图标，将当前画面设为该视频的缩略图。
✏️ 重命名：修改文件名。
🗑️ 删除：彻底删除文件。
文件夹：点击“新建文件夹”整理素材；支持为文件夹设置封面图。
3.4 待机背景（背景页面）
作用：当没有视频播放时，大屏显示的默认图片（如活动主视觉）。
设置：上传图片后，点击图片即可设为当前待机图。
